# Life on Land (SDG 15) Website 🌍

A responsive, multi-page web project dedicated to raising awareness about the United Nations Sustainable Development Goal 15: Life on Land. Built entirely from scratch using fundamental web technologies.

## 🌟 Key Features
* **Multi-Page Architecture:** Includes Home, Content pages, Tables, and Contact sections with seamless navigation.
* **Responsive Design:** Custom CSS styling to ensure the content looks great across different screen sizes.
* **Form Validation:** Includes a dedicated validation page (`validation_ST2.html`) to handle user input securely.
* **Page Editor:** An interactive page editor (`pageEditor_ST2.html`) allowing dynamic content manipulation.

## 🛠️ Tech Stack
* **Markup:** HTML5
* **Styling:** CSS3 (Custom styles for every individual page)
* **Interactivity:** Vanilla JavaScript
